  1. Commandes

mode multimédia

Une fois le mode multimédia activé :


Les commandes :

/* NAVIGATION MENUS */


- Menu principal

- Menu film

- Menu séries

- Menu musique - Menu musique par artistes - Menu musique par genres

- Menu météo

- Menu photos, Menu images


/* MODE D'AFFICHAGE */


- Affichage en mode {fanart - liste - grande liste - miniature}                        

--> Bascule le mode d'affichage.



/* DEPLACEMENT DANS LES MENUS */


- gauche - droite - en bas - en haut - descend - monte - valider - ok - sélectionner - entrer - retour


/* REGLAGE DU SON */


- coupes / désactives le {volume ou son}

- remets / actives le {volume ou son}

- règle le {volume ou son} à x %


/* COMMANDES (PLAY-PAUSE-STOP-NEXT-PRECEDENT) */


- lire / lecture / play

- pause

- stop / arrête

- suivant / suivante

- précédent / précédente


/* METS LA RADIO ( voir fichier xml pour ajouter des radios ) */


- met la radio xxx      

--> mot clé radio


/* LECTURE DE LA MUSIQUE SELON (ARTISTE-TITRE-ALBUM) */


- Je veux écouter l'artiste ASAF AVIDAN      

--> mot clé artiste

- Je veux écouter le titre DIFFERENT PULSE  

--> mot clé titre

- Je veux écouter l'album hors contrôle  

--> mot clé album


 /* LECTURE D'UNE SERIE (TITRE) + FILM PAR TITRE */


- Je veux {regarder ou voir} le film XXX                      

--> mots clé {regarder ou voir} + film

- Je veux {regarder ou voir} la série WESTWORLD              

--> mots clé {regarder ou voir} + série / Le plugin détecte les épisodes déjà vues et reprend l'épisode en cours...


/* SCROLLING */


- scrolling / scrolling arrière              

--> Si vous êtes dans le menu film, la fonction scrolling vous permet de faire défiler automatiquement la liste. La vitesse de défilement est

                                               paramétrable dans Avatar ("speed_scrolling": "1500")


/* FILMS A VOIR */


- liste moi les films non vu                        

--> Énonce les films non vu


/* RECHERCHE ITEM */


- Recherche ou affiche XXX                                

--> Se place directement sur l'item recherché.


/* MISE A JOUR DES LIBRAIRIES */


- update médiathèque musique

- update médiathèque vidéo


/* QUITTER LE MODE MULTIMEDIA */


- Quitter
